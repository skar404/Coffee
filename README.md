# Coffee morse

Text to coffee morse code and coffee morse to text

Install:
```bash
pip install coffee-morse
```

Use: 
```bash
> coffee -t 'coffee'
COFFEE coffee
COFFEE COFFEE COFFEE coffee
COFFEE coffee COFFEE
COFFEE coffee COFFEE
COFFEE coffee coffee
COFFEE coffee coffee

> coffee -m 2 -t 'COFFEE coffee
COFFEE COFFEE COFFEE coffee
COFFEE coffee COFFEE
COFFEE coffee COFFEE
COFFEE coffee coffee
COFFEE coffee coffee'

coffee
```
